== 痛点

由于数量实在是太多， 一个一个单独执行 `docker rmi -f xxx` 肯定是搞不定的。

== 解决方案

删除虚悬镜像的方式是通过

`docker rmi $(docker images -a -f "dangling=true" -q)` 解决的。

是否可以通过 `docker images -a -f "REPOSITORY=xxx -q"`解决呢？ 结果不行。

于是通过

[source,bash]
----
──> docker images | grep android-box | awk '{print $3}' //<1>
